# Support

Use a GitHub issue for reproducible PulsePlayer bugs. Include:

- PulsePlayer version or commit;
- Xcode/Swift version and iOS/tvOS version;
- device or Simulator model;
- sanitized media type and URL shape;
- minimal code and relevant `PlayerEvent`/`PlayerError` output.

Use GitHub Discussions, when enabled, for integration questions and design
proposals. Security issues belong in the private process in `SECURITY.md`.

The project is maintained on a best-effort basis. Private application debugging,
FairPlay server operation, CDN configuration, and guaranteed response times are
not included.
